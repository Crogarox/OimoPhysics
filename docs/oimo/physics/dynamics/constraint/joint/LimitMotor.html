API Documentation

class LimitMotor

package oimo.physics.dynamics.constraint.joint

A limit motor has information of a limit and a motor.

Constructor

new ()

Variables

lowerLimit:Float

The lower bound of the limit. The limit is disabled if lowerLimit > upperLimit. If the limit is translational, the unit of lowerLimit is usually meters. If the limit is rotational, the unit of lowerLimit is usually radians.

maxMotorAcceleration:Float

The maximum acceleration of the motor. This indirectly determines the maximum force/torque of the motor. The motor is disabled if maxMotorAcceleration <= 0. If the motor is translational, the unit of motorSpeed is usually meters per second squared. If the motor is rotational, the unit of motorSpeed is usually radians per second squared.

motorSpeed:Float

The target speed of the motor. If the motor is translational, the unit of motorSpeed is usually meters per second. If the motor is rotational, the unit of motorSpeed is usually radians per second.

upperLimit:Float

The upper bound of the limit. The limit is disabled if lowerLimit > upperLimit. If the limit is translational, the unit of upperLimit is usually meters. If the limit is rotational, the unit of upperLimit is usually radians.

Methods

clone ():LimitMotor

Returns a clone of the object.

setLimits (lower:Float, upper:Float):LimitMotor

Sets limit properties at once and returns this. this.lowerLimit is set to lower, and this.upperLimit is set to upper.

setMotor (speed:Float, maxAcceleration:Float):LimitMotor

Sets motor properties at once and returns this. this.motorSpeed is set to speed, and this.maxMotorAcceleration is set to maxAcceleration.
