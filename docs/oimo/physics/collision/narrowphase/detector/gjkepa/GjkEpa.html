Methods

inline computeClosestPoints (c1:ConvexShape, c2:ConvexShape, tf1:Transform, tf2:Transform, cache:CachedDetectorData):Int

Computes the closest points of two convex shapes c1 and c2 with the transformations tf1 and tf2,and returns the status of the result (see GjkEpaResultState for details). If cached data cache is not null, this method tries to improve performance using the previous result stored in cache, and stores the new result to cache.

Set the compiler option -D OIMO_GJK_EPA_DEBUG for debugging (warning: massive logging).
